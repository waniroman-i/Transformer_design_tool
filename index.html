// EQUATIONS USED IN THIS TOOL

  • Ap= (V1 · I1) / (k_f · f · Bm · J · Kw) × 10¹² [mm⁴]Area product — selects minimum core size
  • N1= V1 / (k_f · f · Bm · Ac) → ceilPrimary turns from Faraday's law
  • N2= V2 / (k_f · f · Bm · Ac) → ceilSecondary turns
  • A_wire= I / J [mm²]Minimum conductor cross-section; J = 3.5 A/mm² (copper)
  • Kw′= (N1·A1 + N2·A2) / AwActual window utilisation — must be ≤ 0.29
  • Paralleln = ceil(A_required / A_single_wire)Number of wires in parallel when no single SWG is large enough
Constants: Bm = 0.25 T (ferrite) · J = 3.5×10⁶ A/m² · Kw_target = 0.29
Sine: k_f = 4.44  |  Square: k_f = 4.0
